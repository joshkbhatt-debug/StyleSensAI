# StyleSensei Extension Icons

## Required Icon Files

You need to create the following icon files in this directory:

- `icon16.png` - 16x16 pixels
- `icon32.png` - 32x32 pixels  
- `icon48.png` - 48x48 pixels
- `icon128.png` - 128x128 pixels

## Icon Design

The StyleSensei icon should:
- Feature a star/burst design (like the SVG in the UI)
- Use the StyleSensei brand colors: Blue (#3B82F6) to Purple (#8B5CF6)
- Have a clean, professional look
- Be recognizable at small sizes

## Quick Icon Creation

You can:
1. Use an online icon generator
2. Create them in Figma/Sketch/Photoshop
3. Use the SVG from the UI as a base:
```svg
<svg viewBox="0 0 24 24" fill="none">

```
