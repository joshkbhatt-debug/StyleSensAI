StyleSensei

AI Writing Assistant

How to use StyleSensei:

  • Select any text on any website
  • Click the StyleSensei icon that appears
  • Choose your desired tone and action
  • Copy or replace the improved text

Or: Right-click selected text → "Transform with StyleSensei"

Get your OpenAI API Key:

Visit platform.openai.com → Create account → API Keys → Create new key
